# 面向世界书条目贡献者的编写规则

## 声明

> [!IMPORTANT]
> `raw_entries/` 目前不再接受贡献，请不要修改其中的文件。

## 如何贡献

**使用 Git**：

首先 `Fork` 本仓库

```bash
git clone [your-repo-url] # 克隆 Fork 项目到你的本地
git checkout foo # 创建并切换到 foo 分支（或其他不是 main 的名字），开始编写
git add [filename] # 编辑完条目文件之后，暂存文件
git commit -m "简要描述你的编辑" # 提交更改
git push origin # 提交到远程仓库
```

最后提交你的 Pull Request
