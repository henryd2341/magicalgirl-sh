# 面向世界书条目贡献者的编写规则

## 关键禁止事项

1. 不要为非 `{{user}}` 的男性角色编写条目
2. 禁止深渊内容
3. 禁牛、卖、公交车
4. 禁止抖机灵绕过规则

## 如何贡献

**使用 Git**:

```bash
git clone https://github.com/henryd2341/magicalgirl-sh # 克隆本项目到你的本地
git checkout foo # 创建并切换到 foo 分支，开始编写
git add raw_entries/ # 编辑完条目文件之后，暂存文件
git commit -m "简要描述你的编辑" # 提交更改
git push origin # 提交到远程仓库
```
