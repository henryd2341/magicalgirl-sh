# 面向世界书条目贡献者的编写规则

## 声明

> [!IMPORTANT]
> 编辑 `raw_entries/` 下的文件，即代表您自愿遵守本文档当前和后续列出的所有规则和禁止事项。
> 您理解并同意，项目主导者有权决定您是否有权限再分发和更改 `raw_entries/` 下的文件，以及是否合并您提交的 Pull Requests。
> 您理解并同意，一旦您的 Pull Requests 得到合并，您提交过的所有更改都可以被其他贡献者更改、删除和追加，并覆盖您先前的更改。

## 关键禁止事项

1. **不要**为非 `{{user}}` 的男性角色编写条目，或细化描述
2. 禁止深渊内容
3. 禁牛、卖、公交车
4. 禁止抖机灵绕过规则

## 如何贡献

**使用 Git**：

首先 `Fork` 本仓库

```bash
git clone [your-repo-url] # 克隆 Fork 项目到你的本地
git checkout foo # 创建并切换到 foo 分支（或其他不是 main 的名字），开始编写
git add raw_entries/ # 编辑完条目文件之后，暂存文件
git commit -m "简要描述你的编辑" # 提交更改
git push origin # 提交到远程仓库
```

最后提交你的 Pull Request
